Curso de CSS Basico

CSS es un lenguaje de hojas de estilo utilizado para darle presentación a sitios web creados con HTML, después de haber aprendido HTML lo siguiente que tienes que aprender es CSS, la definición anterior puede parecer un poco confusa pero te explico, CSS es el lenguaje que nos permite darle estilos, formas, tamaños, colores y posición a los elementos creados en un sitio web.

Docente: Juan Jose Miranda

Fecha de Inicio: 15 de Noviembre del 2021
Fecha de Finalizacion: 26 de Diciembre del 2021
Horario: Lunes a Viernes de 9-11 am

Temas del Curso

  • Maquetación básica
  • Fuentes y texto
  • Color y fondos
  • Listas
  • Bordes
  • Vincular los estilos CSS a páginas HTML
  • Hojas de estilo CSS inline
  • Hojas de estilo CSS internas
  • Hojas de estilo CSS externas
  • Haciendo estructura HTML de la página
  • Agregando estilos del contenedor
  • Estableciendo estilos para el div,header,main, aside y footer
  • Estableciendo estilos y el css grid para el header y el contenido (main)
  • Agregando algunos estilos al sidebar y colocandole CSS Grid.
  • Estilos de los widgets
  • Agregando estilos y CSS grid al footer
  • Estableciendo grid-area

Objetivos del Curso

  • Ser capaz de construir sitios web a través CSS.
  • Estudiar conceptos básicos de CSS y las nuevas técnicas de CSS3 ya disponibles para presentación y estructuración.
