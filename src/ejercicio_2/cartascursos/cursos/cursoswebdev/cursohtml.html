Curso de HTML Basico

En este curso aprenderán los fundamentos de este lenguaje de marcado llamado HTML, el cual se utiliza para crear sitios web, asi que asi no te preocupes estas en el lugar correcto para aprender desde lo mas básico y fundamental para crear paginas web.

Docente: Juan Armando Arrieta

Fecha de Inicio: 1 Diciembre 2021
Fecha de Finalizacion: 15 de Enero del 2022
Horario: Lunes a Viernes de 7-10 pm

Temas del Curso

  • Introducción: internet y cómo se transmite la información.
  • ¿Qué es y para qué sirve HTML?
  • ¿Qué necesito para escribir código HTML y crear páginas web?
  • Estructura básica de una página HTML. HEAD, BODY.
  • Etiquetas HTML básicas: formato texto, imágenes, encabezados, párrafo, saltos de línea, separadores y comentarios.
  • Hipervínculos en HTML.
  • Listas en HTML.
  • Tablas en HTML.
  • Formularios en HTML. Métodos de envío: get y post.
  • Capas en HTML. Etiquetas DIV y SPAN. Maquetación de estructura de páginas web.
  • Página principal. Cabecera, menú, cuerpo y pie.
  • Permitir selección de archivos para envío a través de un formulario.

Objetivos del Curso

  • Conocer cuál es la base de la programación de páginas Web.
  • Dominar el lenguaje de marcación HTML.
  • Etiquetas y atributos. Utilizar etiquetas para definir los formatos de las páginas Web, los textos que las componen, imágenes, listas y esquemas, hipervínculos, marcadores, tablas, páginas de marcos, formularios y elementos decorativos.
  • Incluir hojas de estilo en un página Web.
  • Aprender la definición de clases y capas.
