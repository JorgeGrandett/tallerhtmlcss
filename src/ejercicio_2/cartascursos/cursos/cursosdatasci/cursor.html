Curso de R

En este curso práctico, aprenderá cómo programar en R y cómo usar R para un análisis y visualización de datos efectivos y cómo hacer uso de esos datos de una manera práctica. Aprenderá a instalar y configurar el software necesario para un entorno de programación estadística y a describir conceptos de lenguaje de programación genéricos a medida que se implementan en un lenguaje estadístico de alto nivel.
